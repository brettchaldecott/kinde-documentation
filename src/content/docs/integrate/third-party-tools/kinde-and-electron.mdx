This guide shows you how to integrate **Kinde authentication** into a desktop app built with **Electron**. You’ll set up a basic Electron project, configure Kinde’s SDK, and implement login, logout, and user profile access.

## What you need

- Latest version of Node.js installed on your computer
- A [Kinde](https://kinde.com/) account (Sign up for free)

## Step 1: Create a Kinde application

1. On the Kinde home page, select **Add application.**
    1. Enter a name for your application.
    2. Select **Back-end web** as the type.
    3. Select **Save**. The **Quick start** page opens.
2. In the **Quick Start** section, select **Node.js**, and then select **Save** to continue.
3. On the **Starter kit** tab, select **Set** for both  **Set callback URL** and **Set logout URL** for your application.

![Set callback and logout url](https://imagedelivery.net/skPPZTHzSlcslvHjesZQcQ/98f4af4c-cb40-423e-dc87-81549161e800/public)

1. Scroll down and make a copy of the **environment variables**—you’ll need them in the next section.

![env variables - electron app](https://imagedelivery.net/skPPZTHzSlcslvHjesZQcQ/de8eb34e-0e86-4e20-0cf9-34dddfc59600/public)

## Step 2: Initialize an Electron app

1. Open a terminal and run the following commands to create a new folder for your Electron app, and then navigate into it.
    
    ```bash
    mkdir electron-kinde-auth
    cd electron-kinde-auth
    ```
    
2. Create a new `package.json` file:
    
    ```bash
    touch package.json
    ```
    
3. Open the newly created `package.json` file with your code editor and add the following:
    
    ```json
    {
      "name": "electron-kinde-auth",
      "description": "A simple Electron app to authenticate with Kinde using OAuth",
      "productName": "Electron Kinde Auth",
      "version": "1.0.0",
      "main": "main.js",
      "scripts": {
        "start": "electron ."
      },
      "keywords": [],
      "author": "",
      "license": "MIT"
    }
    
    ```
    
4. Run the following command to install the required packages:
    
    ```bash
    npm install @kinde-oss/kinde-node-express dotenv express pug
    ```
    
5. Install the `electron` package as development dependency:
    
    ```bash
    npm install --save-dev electron
    ```
    
6. Create a `.env` file for environment variables:
    
    ```bash
    touch .env
    ```
    
7. Open it in your editor and paste the Kinde credentials copied from the Quick Start page.
    
    ```bash
    KINDE_ISSUER_URL=https://<KINDE_DOMAIN>.kinde.com
    KINDE_CLIENT_ID=<CLIENT_ID>
    KINDE_CLIENT_SECRET=<CLIENT_SECRET>
    KINDE_SITE_URL=http://localhost:3000
    KINDE_LOGOUT_REDIRECT_URI=http://localhost:3000
    KINDE_DOMAIN=https://<KINDE_DOMAIN>.kinde.com
    KINDE_REDIRECT_URI=http://localhost:3000/callback
    ```
    
8. Create a new `server.js` file:
    
    ```bash
    touch server.js
    ```
    
9. Add the following code into the `server.js` file. 
    
    This file:
    
    - imports all the required dependencies.
    - creates the Express.js server for authentication.
    - creates a Kinde client to authenticate users.
    - creates the root and admin routes for the application.
    
    ```jsx
    require("dotenv").config()
    const express = require("express")
    const path = require("path")
    const {
      setupKinde,
      protectRoute,
      getUser,
      GrantType,
    } = require("@kinde-oss/kinde-node-express")
    
    const app = express()
    const port = 3000
    
    const config = {
      clientId: process.env.KINDE_CLIENT_ID,
      issuerBaseUrl: process.env.KINDE_ISSUER_URL,
      siteUrl: process.env.KINDE_SITE_URL,
      secret: process.env.KINDE_CLIENT_SECRET,
      redirectUrl: process.env.KINDE_REDIRECT_URI,
      postLogoutRedirectUrl: process.env.KINDE_LOGOUT_REDIRECT_URI,
      unAuthorisedUrl: process.env.KINDE_SITE_URL,
      grantType: GrantType.AUTHORIZATION_CODE,
    }
    
    const client = setupKinde(config, app)
    
    app.set("view engine", "pug")
    app.set("views", path.join(__dirname, "views"))
    app.use(express.static(path.join(__dirname, "public")))
    
    app.get("/", async (req, res) => {
      if (await client.isAuthenticated(req)) {
        res.redirect("/admin")
      } else {
        res.render("index", { title: "Welcome", message: "Please log in" })
      }
    })
    
    app.get("/admin", protectRoute, getUser, (req, res) => {
      res.render("admin", { title: "Dashboard", user: req.user })
    })
    
    app.listen(port, () => {
      console.log(`✅ Express server running on http://localhost:${port}`)
    })
    ```

16. Run the following command to create your Electron entry point:
    
    ```bash
    touch main.js
    ```

17. Paste the following code into `main.js`.
    
    This script will:
    
    - Launch a new Electron app window.
    - Spawn a Node child process to start the Express server you set up earlier.
    
    ```jsx
    const { app, BrowserWindow } = require("electron")
    const path = require("path")
    const { spawn } = require("child_process")
    
    let serverProcess
    let win
    
    function createWindow() {
      win = new BrowserWindow({
        width: 1000,
        height: 770,
        webPreferences: {
          nodeIntegration: false,
          contextIsolation: true,
        },
      })
    }
    
    app.whenReady().then(async () => {
      serverProcess = spawn("node", [path.join(__dirname, "server.js")], {
        cwd: __dirname,
        shell: true,
        env: process.env,
        stdio: "inherit",
      })
    
      createWindow()
      try {
        win.loadURL("http://localhost:3000")
      } catch (err) {
        console.error("❌ Server error:", err)
        win.loadURL("data:text/html,<h1>❌ Failed to start server</h1>")
      }
    })
    
    app.on("window-all-closed", () => {
      if (serverProcess) serverProcess.kill()
      if (process.platform !== "darwin") app.quit()
    })
    ```

    ```bash
    npm start
    ```
